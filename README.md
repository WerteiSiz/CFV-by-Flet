

# САПР VSAT "CFV"

**Система автоматизированного проектирования спутниковых линий связи**

САПР VSAT — это десктопное приложение, разработанное на Flet (Python), которое позволяет выполнять инженерные расчёты для спутниковых систем связи: позиционирование антенн, энергетические расчёты линий «вверх» и «вниз», а также поддержку принятия решений по выбору оборудования.

## Возможности системы

- Точный расчёт угла места, азимута и наклонной дальности для позиционирования антенны.
- Энергетический расчёт линии «Земля — Спутник» (Uplink) с учётом атмосферных потерь.
- Энергетический расчёт линии «Спутник — Земля» (Downlink) с визуализацией.
- Интеллектуальная система поддержки принятия решений для подбора оптимального диаметра рефлектора.
- Экспорт результатов расчётов в форматы TXT, Word и Excel.
- Современный графический интерфейс с адаптивными карточками и навигацией.

## Технологии

- **Python 3.10+**
- **Flet** — фреймворк для создания нативных интерфейсов
- **Pandas, Scikit-learn** — для машинного обучения (СППР)
- **python-docx, openpyxl** — экспорт в Word и Excel
- **Math, Datetime** — математические расчёты и работа с датами

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-username/cfv-by-flet.git
cd cfv-by-flet/papka
```

### 2. Создание виртуального окружения (рекомендуется)

```bash
python -m venv .venv
```

Активация:
- Windows: `.venv\Scripts\activate`
- Linux/macOS: `source .venv/bin/activate`

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

Если файла `requirements.txt` нет, установите вручную:

```bash
pip install flet pandas scikit-learn python-docx openpyxl
```

### 4. Проверка структуры папок

Убедитесь, что в папках `components` и `pages` присутствуют файлы `__init__.py` (пустые) — это необходимо для корректных импортов.

## Запуск приложения

```bash
python ui.py
```

После запуска откроется окно приложения с верхней панелью навигации и главной страницей.

## Использование

### Навигация
- Клик по пунктам меню («САПР VSAT», «Позиционирование», «ЗС-ИЗС», «ИЗС-ЗС», «СППР») переключает соответствующие разделы.

### Раздел «Позиционирование»
- Введите долготу, широту и точку стояния спутника (в градусах).
- Нажмите «Произвести расчёты» — появятся угол места, азимут и наклонная дальность.
- Кнопка «Сохранить» позволяет экспортировать результаты в TXT, Word или Excel.

### Разделы энергетических расчётов
- Задайте частоту, диаметр антенны, коэффициент использования, а также метеоданные (давление, температуру, влажность).
- После расчёта отображаются потери, усиление и итоговая мощность сигнала.
- Результаты можно сохранить.

### СППР
- На основе введённых параметров система предложит оптимальный диаметр рефлектора, используя обученную модель.

## Экспорт результатов

Все результаты можно сохранить в трёх форматах:
- **TXT** — простой текст
- **Word** — отформатированный документ с таблицами
- **Excel** — файл с возможностью дальнейшего анализа

При успешном сохранении появляется всплывающее уведомление.

## Разработка и доработка

- Для добавления новых страниц создайте файл в папке `pages` и пропишите функцию `show(container, page)`.
- Цветовую схему можно изменить в `components/constants.py` (словарь `COLORS`).
- Математические модели для СППР обучаются отдельно, но можно загрузить готовую через `components/ml.py`.
