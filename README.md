## Запуск проекта чере командную строку
Нажмите Win + R, введите cmd, перейдите в папку проекта:


```cd C:\TIP\projects\CFV-by-Flet```

Создайте и активируйте окружение:

```python -m venv .venv```

```.venv\Scripts\activate.bat ```

Установите зависимости и запустите:

```pip install -r requirements.txt``
python main.py```
